# Next.js Mistral Demo Chat Web App
*Created by*: Stan van Baarsen

October 2024

## Installation
Step 1:
⁠```bash
nvm use
npm install
```

Step 2: 
In the .env.local file, add a line with
```
⁠API_KEY=YOUR_MISTRAL_API_KEY
```

## Usage
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to open the web app!
